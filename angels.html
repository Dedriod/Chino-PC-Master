Código Maestro

Se guarda en este navegador (local) y se usa para crear, listar y editar todos los proyectos.

Quién va en qué cuenta

  • ldcalvoe4595@gmail.com — solo el Apps Script Maestro (angels_maestro/Código.js): registro de proyectos, hashes, Drive e imágenes de cabecera.
  • Cuenta del evento (u otra) — el Apps Script Emisor (angels_sender): Gmail, hojas del proyecto, cola de envíos. Puedes enviarle el código desde la sección «Script del Emisor» (por correo o Drive).

Enlaces públicos (sin login en esta web)

  • Participantes — URL con #u/<hash>: acceso solo con el enlace, sin contraseña del sitio.
  • Organizador — URL con #a/<hash>: acceso solo con el enlace + la contraseña de organizador que definiste al crear el proyecto (no es la sesión de Chino PC Master).

Configurar el Maestro en Google

Un solo proyecto Apps Script «Maestro» y una sola hoja de cálculo sirven para todos los eventos Ángeles; no hace falta duplicar script ni spreadsheet por cada proyecto web.

  1. ID del script (para Clasp): en script.google.com, abre el proyecto Maestro → engranaje «Configuración del proyecto» → «ID de secuencia de comandos». Ese valor va en fuentes-apps-script/angels_maestro/.clasp.json como scriptId. Desde esa carpeta en consola: clasp login (misma cuenta), luego clasp push. Requisito previo: en Ajustes del usuario, activar la API de Apps Script.
  2. Propiedades del script (mismo menú de configuración → «Propiedades del script»; no van en el código del repo): SPREADSHEET_ID = el ID que aparece en la URL de tu Sheet (…/spreadsheets/d/ID/…); MASTER_ADMIN_PIN = una contraseña que tú eliges (Google no la genera); la misma cadena usarás en el asistente web como PIN de super admin.
  3. Implementación → Aplicación web → Ejecutar como tú, acceso Cualquiera. Copia la URL que termina en /exec.
