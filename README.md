# 🧧 Chino PC Master - Sistema de Rifas Digitales

¡Bienvenido al ecosistema de **Chino PC Master**! Esta es una solución web profesional diseñada para la gestión y automatización de rifas digitales, conectada en tiempo real con Google Sheets.

## 🚀 Características Principales
- **Gestión en Tiempo Real:** Conexión directa con Google Apps Script.
- **Interfaz Profesional:** Diseño optimizado para soporte técnico y ventas de tecnología.
- **Panel de Configuración:** Edición dinámica de premios, precios y colores desde el navegador.
- **Arquitectura Eficiente:** Carga de assets locales para máxima velocidad y disponibilidad.

## 🛠️ Stack Tecnológico
- **Frontend:** HTML5, CSS3 (Custom Properties), JavaScript (ES6+).
- **Backend:** Google Apps Script (V8 Engine).
- **Base de Datos:** Google Sheets API.
- **Hosting:** GitHub Pages.

## 📂 Estructura del Proyecto
- `/img`: Logos, iconos de WhatsApp y assets visuales del sitio.
- `/css`: Estilos personalizados de la marca.
- `/js`: Lógica de la rifa y conexión con el API de Google.
- `index.html`: Punto de entrada principal de la aplicación.
